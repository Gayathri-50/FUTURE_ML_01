# Sales & Demand Forecasting

## Objective
Predict future sales using historical business data.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- VS Code

## Workflow
1. Data Loading
2. Data Cleaning
3. Date Conversion
4. Feature Engineering
5. Visualization
6. Forecasting

## Model
Linear Regression

## Evaluation
- MAE
- RMSE
- R² Score

## Results
Generated actual vs predicted sales graph and 7-day future forecast.

## Business Insights
- Historical sales trends identified
- Forecast supports future planning
- Visualization improves decision making
